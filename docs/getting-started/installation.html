Installation

Requirements:

  • Git
  • PNPM
  • AWS CDK Toolkit
    • pnpm add -g cdk
  • Bootstrapped AWS Account
    • cdk bootstrap aws://<account-number>/<region>
  1. Setup environment
pnpm env use --global 16
pnpm add -g gboost
info

Don't want to globally install? use: pnpm dlx gboost ... wherever you see gboost

  1. Scaffold your web app
gboost create # follow prompts
cd my-gb-app
pnpm i # install dependencies
note

Make sure your AWS Account has been bootstrapped with the AWS CDK and your AWS CLI credentials are present in the terminal before you run the next command

  1. Boostrap your AWS account and deploy infrastructure
gboost deploy-dev
  1. Visit the CloudFront URL logged by gboost deploy-dev
